# Checkly Support Site

Support and privacy policy pages for **Checkly — Calendar Checklist**, an iOS
app that combines your calendar and to-do list: create events with automatic
reminders, check them off like tasks, and get an urgent alert if a to-do
slips past its deadline.

Served via GitHub Pages:

- **Support page**: https://helenminxu.github.io/checkly-support/
- **Privacy policy**: https://helenminxu.github.io/checkly-support/privacy.html
